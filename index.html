USAGE

Login

[USERNAME] [PASSWORD]

At the moment CUD only supports authentication with username and password. Public-Key Authentication will be added soon.

example:
  • testuser testilo

Create - start a new server

create [GAMETYPE]

This creates a new gameserver. Run create help for a list with all supported gametypes and their options.

example:
  • create minecraft-vanilla

Create with options

You can add further more options as you want. All datatypes, like Integer or Boolean, can easily be appended with a equality sign at the option.

example:
  • create minecraft-vanilla cmd=true world="WaterHome Front"

Plugins - Do Ur thing!

Of course, CUD also supports minecraft plugins :P

example:
  • create spigotmc cmd=true level-type=FLAT plugin=WorldEdit max-mem=10G

shortcut - save time

Shortcuts are equal with a normal command. Except that they are shorter. In the example below, the shortcut "the-dropper-2" is converted into the following create minecraft-vanilla map="The Dropper 2" cmd=true view=15. You save time with those. Especially when you don't know the full command or start those types of server very often (e.g. in development).

example:
  • shortcut the-dropper-2
